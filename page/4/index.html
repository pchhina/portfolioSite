About

I work at Gamma Technologies where I help engineers at Ferrari, BMW, Ford, US Air Force and many others design better vehicles. I do that through developing and supporting GT-SUITE, a powerful system-level simulation software, that allows our customers to evaluate their design ideas accurately before making any test prototypes. Please see my linkedin account if you are interested in learning more about this area of my work.

In my spare time, I learn and try to build things using open source tools. The powerful nature of these tools and the awesome community building them has kept my curiosity growing to understand data in useful and meaningful ways. All my work on this website is open source and available on Github. Please feel free to look through my projects. I would appreciate any critique and feedback.

Quizme

This project was inspired by a book I read a few years back. It is a command line tool to create repository of questions and automatically schedule daily quizzes for deliberate practice based on spaced repetetion algorithm. It employs data.table to store questions and answers that enables fast re-ranking of questions based on user’s responses. Since questions and answers are simple text data, you could use it to learn almost anything.

  • Date: 2018-10-18
  • Category: General application

Guess the Number

This is a fun application to explore and learn Tkinter, a python library that interfaces with Tk, a library for building GUIs. The game itself is simple - you try to guess a number between 1 and 100 within 7 trials, but it showcases the use of few elements of graphical user interfaces - buttons, data entry fields and labels. Feel free to check this out at my github account.

Stopwatch

A stopwatch built with python library, tkinter. Time is displayed to tenth of a second precision. Beyond the basic stopwatch functionality, a game-mode can be toggled on/off with a checkbox at the bottom. This allows the user to play a game where a player scores a point each time they could stop on a whole second.

  • Client: Personal (See Code on Github)
  • Date: 2019-06-13
  • Category: Interactive Programming, Games

Pong

Game of pong built using Tkinter. If the ball hits the gutter instead of paddle, the opponent scores a point. If the player is successful at blocking the ball with paddle, the ball bounces off the paddle towards the opponent at a slightly higher speed. Key pairs (f, d) and (j, k) are used to move both paddles to make contact with the ball.

  • Date: 2019-06-30
